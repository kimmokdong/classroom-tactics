# TFT 시즌 17 (우주의 신들) - 시너지 구간별 효과 총정리

TFT 세트 17 "우주의 신들(Space Gods)"에 등장하는 모든 시너지의 구간별 효과를 정리한 문서입니다. 시너지는 **기원(Origin)**과 **직업(Class)**으로 나뉘며, 각 시너지는 특정 수의 챔피언을 배치할 때 활성화됩니다.

> **참고**: 수치는 라이브 서버(패치 17.1~17.2) 기준이며, 패치에 따라 변동될 수 있습니다.

---

## 기원 (Origins)

### 동물특공대 (Anima)

플레이어 전투에서 패배하면 기술(Tech) 20을 획득하며, 연패 횟수 × 5만큼 추가 획득합니다. 동물특공대 처치 관여 시 기술 2를 추가 획득합니다. 기술이 100에 도달할 때마다 동물특공대 무기를 제작합니다.

| 구간 | 효과 |
|------|------|
| (3) | 연구 시작 - 기술 축적 개시 |
| (6) | 플레이어 전투 후 전리품 획득 |

**챔피언**: 브라이어(1), 징크스(2), 오로라(3), 일라오이(3), 피오라(5)

### 중재자 (Arbiter)

고유 신성 법률을 작성하여, 선택한 원인(cause)이 발생할 때 중재자에게 효과를 적용합니다.

| 구간 | 효과 |
|------|------|
| (2) | 원인과 효과를 선택하여 법률 작성 |
| (3) | 효과가 더 강력해짐 |

**챔피언**: 레오나(1), 조이(2), 다이애나(3), 르블랑(4)

### 암흑의 별 (Dark Star)

블랙홀을 생성하여 최대 체력 10% 이하인 적을 즉시 처형합니다.

| 구간 | 효과 |
|------|------|
| (2) | 블랙홀 생성 - 10% 체력 이하 적 처형 |
| (4) | 추가로 공격력 & 주문력 30% 증가 |
| (6) | 가장 강한 암흑의 별 유닛이 초거대화, 암흑의 별 효과 85~100% 적용 + 소형 블랙홀 2개 생성 |
| (9) | 모든 암흑의 별 유닛이 초거대화 |
| (Lv.10 프리즘) | 모든 적을 집어삼김 |

**챔피언**: 초가스(1), 리산드라(1), 모데카이저(2), 카이사(3), 카르마(4), 진(5)

### 별돌보미 (Stargazer)

매 게임마다 무작위 별자리가 배정됩니다. 강화된 칸에 있는 아군이 별자리에 따른 효과를 획득합니다. 플레이어 레벨이 오를수록 더 많은 강화 칸이 공개됩니다.

#### 별자리: 뱀자리 (The Serpent)
아군에게 내구력 부여, 별돌보미는 추가 내구력 + 적 중독 (입힌 피해의 일부를 3초간 마법 피해로 반복)

| 구간 | 효과 |
|------|------|
| (3) | 내구력 5~15%, 독 피해 30% |
| (5) | 내구력 10~20%, 독 피해 45% |
| (7) | 내구력 15~25%, 독 피해 60% |

#### 별자리: 사냥꾼자리 (The Huntress)
가장 체력이 높은 적을 표시. 강화 칸 아군 공격 속도 증가, 별돌보미는 표시된 적 처치 시 최대 체력 15% 회복

| 구간 | 효과 |
|------|------|
| (3) | 공격 속도 12~30%, 표시 3개 |
| (5) | 공격 속도 35~45%, 표시 5개 |
| (7) | 공격 속도 55~60%, 표시 7개 |

#### 별자리: 산자리 (The Mountain)
5회 플레이어 전투마다 별돌보미 엠블럼 획득. 강화 칸 별돌보미에게 누적 보너스 부여

| 구간 | 효과 |
|------|------|
| (3) | 체력 15% 증가 |
| (4) | + 공격력 & 주문력 15% |
| (5) | + 방어력 & 마법 저항력 10~15 |
| (6) | + 공격 속도 10~15% |
| (7) | + 내구력 6~12% |
| (8) | + 모든 보너스 10~20% 증가 |
| (9+) | 정상이 보인다... |

#### 별자리: 제단자리 (The Altar)
유닛이 사망할 때마다 제단에 제물로 바쳐짐. 강화 칸 아군 체력 & 공격 속도 증가

#### 별자리: 메달자리 (The Medallion)
강화 칸 아군 피해 증폭 획득, 3성 아군 1명당 추가 증폭

| 구간 | 효과 |
|------|------|
| (3) | 피해 증폭 12~15%, 3성 당 +3~5% |

#### 별자리: 분수자리 (The Fountain)
강화 칸 아군 마나 재생 획득, 별돌보미는 스킬로 가장 체력 낮은 아군 치유

| 구간 | 효과 |

| (3) | 마나 재생 3, 스킬 피해의 10% 치유 |

#### 별자리: 멧돼지자리 (The Boar)

### 습격자 (Marauder)

팀 전체 모든 피해 흡혈 5% 부여. 습격자는 추가 흡혈 + 공격력을 획득하며, 초과 흡혈분이 보호막으로 전환 (최대 체력 25%까지).

| 구간 | 효과 |
|------|------|
| (2) | 흡혈 5%, 공격력 +20% |
| (4) | 흡혈 7%, 공격력 +30% |
| (6) | 흡혈 10%, 공격력 +40% |

**챔피언**: 아칼리(2), 벨베스(2), 우르곳(3), 마스터 이(4), 피오라(5)

---

### 복제자 (Replicator)

복제자의 스킬이 감소된 효과로 한 번 더 발동됩니다.

| 구간 | 효과 |
|------|------|
| (2) | 22~25% 위력으로 재발동 |
| (4) | 45~50% 위력으로 재발동 |

**챔피언**: 리산드라(1), 베이가(1), 판테온(2), 룰루(3), 미스 포츈(3), 나미(4)

### 불한당 (Rogue)

불한당은 공격력 & 주문력을 획득합니다. 체력 50% 이하로 떨어지면 그림자 속으로 사라지며, 적의 타겟이 근처 탱커로 전환됩니다.

| 구간 | 효과 |
|------|------|
| (2) | 공격력/주문력 +12~15% |
| (3) | 공격력/주문력 +25~30% |
| (4) | 공격력/주문력 +40~45% |
| (5) | 공격력/주문력 +55~60% |

**챔피언**: 브라이어(1), 탈론(1), 그웬(2), 피즈(3), 카이사(3), 리븐(4)

---

### 길잡이 (Shepherd)

별의 결속을 소환하여 전투를 돕습니다. 별 레벨 총합에 비례하여 강화됩니다.

| 구간 | 효과 |
|------|------|
| (3) | 비아(Bia) 소환 |
| (5) | 바이엔(Bayin) 소환 |
| (7) | 비아와 바이엔의 결속이 깊어짐 |

**챔피언**: 리산드라(1), 티모(1), 꼬마 정령(2), 일라오이(3), 르블랑(4), 소나(5)

### 저격수 (Sniper)

저격수는 피해 증폭을 획득하며, 대상이 멀수록 증가합니다.

| 구간 | 효과 |
|------|------|
| (2) | 피해 증폭 18% + 칸당 +2% |
| (3) | 피해 증폭 24% + 칸당 +3% |
| (4) | 피해 증폭 28% + 칸당 +4% |
| (5) | 피해 증폭 32% + 칸당 +5% |

**챔피언**: 이즈리얼(1), 나르(2), 사미라(3), 자야(4), 진(5)

### 선봉대 (Vanguard)

선봉대는 보호막을 받는 동안 내구력 5% 증가. 전투 시작 시와 체력 50%에 최대 체력 비례 보호막을 10초간 획득합니다.

| 구간 | 효과 |
|------|------|
| (2) | 최대 체력 16% 보호막 |
| (4) | 최대 체력 28~30% 보호막 |
| (6) | 최대 체력 40% 보호막 + 보호막 중 피해 감소 8~10% |

**챔피언**: 레오나(1), 나서스(1), 모데카이저(2), 일라오이(3), 누누(4), 블리츠크랭크(5)

### 여행자 (Voyager)

전투 시작: 앞줄 2열 아군(탱커/전사)에게 보호막, 뒷줄 2열 아군에게 피해 증폭 부여. 여행자는 2배 적용.

|------|------|

| (6) | 보호막 700 / 피해 증폭 27% |

**챔피언**: 꼬마 정령(2), 파이크(2), 오로라(3), 거대 메크 로봇(4), 카르마(4)
