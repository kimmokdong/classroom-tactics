# 📖 교실 택틱스 (Classroom Tactics) 유닛 스킬 도감

이 문서는 **교실 택틱스**에 등장하는 모든 유닛의 상세 스킬 정보를 담고 있습니다.  
유닛은 코스트(Tier) 별로 분류되어 있으며, 각 유닛의 전용 스킬과 효과 수치를 확인하실 수 있습니다.

---

## 🌟 유닛 정보 읽는 법
- **★ / ★★ / ★★★**: 유닛의 성급에 따른 스킬 수치 변화를 의미합니다.
- **계수 아이콘**: ⚔️ (공격력), 🔮 (주문력), 🛡️ (방어력/마법저항력), ❤️ (최대 체력) 기반 데미지나 효과를 나타냅니다.

---

## 🟢 1코스트 유닛 (Tier 1)

### 🧹 칠판닦이 당번
- **분류**: 도덕 / 선도부
- **스킬**: **일벌백계** (aoe_damage_buff)
- **설명**: 주변 1칸 적에게 피해를 입히고 본인의 방어력을 증가시킵니다.
- **수치**:
  - **피해량**: 공격력(⚔️)의 150% / 220% / 350%
  - **방어력 증가**: 20% / 35% / 50%
  - **지속 시간**: 30 / 40 / 50 틱

### 🏃 달리기 선수
- **분류**: 체육 / 육상부
- **스킬**: **전력질주** (dash_damage)
- **설명**: 가장 먼 적에게 돌진하여 물리 피해를 입히고 기절시킵니다.
- **수치**:
  - **피해량**: 최대 체력(❤️)의 10% / 15% / 25%
  - **기절 지속**: 10 / 15 / 20 틱

### 🔤 영단어 암기왕
- **분류**: 영어 / 장난꾸러기
- **스킬**: **단어 폭격** (random_aoe)
- **설명**: 랜덤 적 3명에게 물리 피해를 입힙니다.
- **수치**:
  - **피해량**: 공격력(⚔️)의 120% / 180% / 300%

### 📖 국어부장
- **분류**: 국어 / 방송부
- **스킬**: **집요한 팩트체크** (single_dot)
- **설명**: 대상에게 맹렬한 팩트체크를 날려 일정 시간 지속적인 마법 피해(도트딜)를 입힙니다.
- **수치**:
  - **총 피해량**: 주문력(🔮)의 300% / 450% / 700%
  - **효과 주기**: 100 / 80 / 60 틱마다 피해 발생

### ➕ 수학 짝꿍
- **분류**: 수학 / 급식부
- **스킬**: **지수 함수 성장** (passive)
- **설명**: 패시브: 매 3회 공격마다 자신의 공격력과 공격 속도가 영구적으로 증가합니다.
- **수치**:
  - **스탯 증가량**: 1% / 2% / 4% (무제한 중첩)

### 🌍 지리덕후
- **분류**: 사회 / 보건부
- **스킬**: **지도 분석** (single_damage)
- **설명**: 대상에게 물리 피해를 입히고 방어력을 관통합니다.
- **수치**:
  - **피해량**: 방어력(🛡️)의 150% / 250% / 400%
  - **방어력 관통**: 30% / 50% / 70%

### 🔭 과학탐구원

- **스킬**: **실험 반응** (single_damage_buff)
- **설명**: 대상에게 물리 피해를 입히고 본인은 피해 흡혈 효과를 얻습니다.
- **수치**:

  - **피해 흡혈**: 10% / 10% / 10%
  - **지속 시간**: 30 / 40 / 50 틱

### 🎵 리코더 요정
- **분류**: 음악 / 급식부
- **스킬**: **힐링 멜로디** (heal)
- **설명**: 체력이 가장 낮은 아군의 체력을 회복시킵니다.
- **수치**:
  - **회복량**: 대상 최대 체력의 20% / 30% / 45%

### 🗿 찰흙 조각가
- **분류**: 미술 / 장난꾸러기
- **스킬**: **점토 방벽** (self_shield)
- **설명**: 본인에게 보호막을 부여합니다.
- **수치**:
  - **보호막량**: 공격력(⚔️)의 150% / 250% / 400%

### 🛑 복도 지킴이

- **스킬**: **출입금지** (taunt)
